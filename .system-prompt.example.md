# System Prompt

This file is appended to Claude's system prompt when handling requests.
Copy this file to `.system-prompt.md` and customize it for your needs.

## Example Usage

You can use this to:
- Set coding style preferences
- Define project-specific conventions
- Add context about your codebase
- Specify response formatting preferences

## Sample Content

```markdown
You are helping with a TypeScript codebase. Follow these conventions:
- Use functional patterns where appropriate
- Prefer explicit types over inference for function signatures
- Write concise, focused responses
```
